## What is MCP?

MCP (Model Context Protocol) is an open standard that lets AI models connect to external tools, databases, APIs, and services. Think of it as **USB-C for AI** — one universal plug that works everywhere.

---

## Who is this for?

- Beginners who want to understand MCP from scratch
- Developers exploring AI tool integrations
- Anyone preparing for AI/MCP roles
